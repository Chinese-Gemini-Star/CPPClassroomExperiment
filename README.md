## 实验心得

通过这次实验,我对多态有了更多地理解,明白了在合理的架构下,使用多态可以大大简化代码的编写.也因此我更多地懂得了合理设置类与类之间关系的意义.

同时,通过对STL库的运用,我对其中的一些常用容器和算法有了更多的认识.比如其中的容器不能把引用类型作为模板参数.
